# Healthcare Analytics Australia – Power BI & SQL Project

## 📊 Project Overview

This end-to-end data analytics project explores the relationship between health conditions and prescription patterns across Australia by integrating multiple public datasets, including Australian Census health data and PBS prescription data.

The goal of this project is to identify patterns in population health, demographic differences, and medication utilisation across Australian states using SQL Server and Power BI.

---

## 🧰 Tools & Technologies

* SQL Server
* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Modelling (Star Schema)
* Excel (data preparation)

---

## 🎯 Objectives

* Integrate multiple public healthcare datasets
* Clean, transform, and structure raw data for analysis
* Build a relational data model using a star schema design
* Develop interactive dashboards in Power BI
* Analyse relationships between health conditions, demographics, and prescription usage

---

## 🗂️ Dataset Sources

* Australian Bureau of Statistics (Census Health Data)
* PBS (Pharmaceutical Benefits Scheme) Prescription Data

---

## 🏗️ Data Model

The project follows a **star schema design**, including:

* Fact table: Health conditions / prescription metrics
* Dimension tables:

  * Age groups
  * Gender
  * Australian states
  * Health conditions
  * ATC medication categories

---

## 🔄 Data Processing Steps

* Data imported into SQL Server
* Data cleaning using SQL and Power Query
* Standardisation of categorical variables
* Handling missing values and inconsistent formatting
* Creation of relationships between tables

---

## 📊 Key Features in Power BI

* KPI cards for key health indicators
* State-level comparison of health conditions
* Analysis of prescription patterns by ATC classification
* Demographic breakdown (age & gender)
* Interactive filters and slicers
* Relationship analysis between conditions and medication usage

---

## 📈 Key Insights

* Significant variation in health conditions across Australian states
* Strong correlation between age groups and prescription volumes
* Nervous system and cardiovascular medications are among the most prescribed categories
* Clear demographic patterns in healthcare utilisation across gender and age groups
